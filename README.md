# MTB

📋 **Едитабилна табела со поддршка за фајлови, линкови и мултимедија**

## 🌐 Demo

👉 **[Отвори ја апликацијата](https://assisstant.github.io/MTB/)**

## ✨ Функции

### 📎 Хибриден Storage систем
- **Локални фајлови** - вградени во HTML (работи offline)
- **Google Drive линкови** - споделување помеѓу компјутери
- **YouTube видеа** - автоматско препознавање
- **Било кој URL** - слики, PDF, документи

### 📊 Табела
- ✏️ Едитабилни ќелии со форматирање (bold, italic, боја)
- 📑 Повеќе страници/табови
- 🖼️ Унифицирани thumbnails за сите типови фајлови
- 📝 Опис за секој прилог

### 💾 Export/Import
- 📄 **HTML** - самостоен фајл со сите податоци
- 📋 **JSON** - backup на податоците
- 🖨️ **PDF** - печатење

### 🎨 Изглед
- ☀️ Светла / 🌙 Темна тема
- 🇲🇰 Македонски / 🇬🇧 English
- 📱 Responsive дизајн
- 🔲 Resizable modal за преглед

## 📖 Како да користиш Google Drive

1. Качи фајл на Google Drive
2. Десен клик → "Share" → "Get link"
3. Избери "Anyone with the link can view"
4. Копирај го линкот
5. Залепи во полето за URL

## 🚀 Инсталација

Само отвори `index.html` во прелистувач - нема потреба од сервер!
